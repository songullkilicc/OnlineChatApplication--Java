# Online Chat Application (Java OOP Project)

This project is a simple client-server online chat application developed in Java.

It was created as a final assignment for the Object-Oriented Programming II course at Kielce University of Technology.

## Features

• Client-Server architecture using TCP sockets  
• GUI desktop client (Java Swing)  
• User login system  
• Real-time text messaging  
• Message storage in files  
• Object-Oriented Programming principles:
  - Encapsulation
  - Inheritance
  - Polymorphism
  - Abstraction

• Generic class usage (MessageStore)
• Custom exceptions
• Unit tests included

## Technologies Used

Java 22  
Java Swing (GUI)  
TCP Networking  
File I/O  
JUnit Testing
